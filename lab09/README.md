# Лабораторна робота № 9

**Тема:** Модульне програмування. Створення багатофайлових проектів та власних бібліотек у мові C++.

**Мета:** Оволодіти навичками розробки програм із модульною структурою. Навчитися створювати та підключати власні заголовкові файли (`.h`) і файли реалізації (`.cpp`). Закріпити практичні навички роботи з операторами розгалуження, циклами, побітовими операціями та створення інтерактивного консольного меню.

**Завдання:**
1. Створити власну бібліотеку (`ModulesZavalai.h` та `ModulesZavalai.cpp`), яка містить прототипи та реалізацію функцій для розв'язання поставлених задач.
2. **Задача 9.1:** Реалізувати функцію для розрахунку суми до сплати за спожиту електроенергію відповідно до заданої багаторівневої системи тарифів.
3. **Задача 9.2:** Реалізувати функцію обробки масиву для розрахунку середньорічної температури за шкалою Цельсія та переведення результату у шкалу Фаренгейта.
4. **Задача 9.3:** Реалізувати функцію для аналізу D10 біта заданого числа і підрахунку кількості двійкових нулів або одиниць у його представленні (з використанням тернарного оператора).
5. **Задача 9.4:** Розробити головний файл програми (`main.cpp`), що містить діалогове меню для зручного виклику всіх реалізованих функцій, включно з функцією `s_calculation` з лабораторної роботи №8.

**Варіант №:** 9
