# 1. Створюємо файл README.md з повним описом лабораторної роботи
cat << 'EOF' > README.md
# Лабораторна робота № 12

**Тема:** Розробка та використання абстрактних типів даних (АТД). Створення та інтеграція користувацьких класів у мові програмування C++.

**Мета:** Навчитися проектувати та реалізовувати абстрактні типи даних у вигляді класів; опанувати механізми інкапсуляції, контролю доступу до даних (public/private), написання конструкторів, методів встановлення (set) та отримання (get) значень; реалізувати автоматизоване модульне тестування (Unit Testing) та підсистему аудиту папок "Teacher" для контролю цілісності структури репозиторію.

**Завдання:** 1. Створити користувацький клас `ClassLab12_Zavalai` для представлення геометричної фігури "Конус" (Варіант 4).
2. Специфікувати інтерфейс класу у заголовковому файлі `ModulesZavalai.h`, а реалізацію методів — у `ModulesZavalai.cpp`.
3. Забезпечити фільтрацію вхідних даних (автоматичне відсікання від'ємних значень радіуса та висоти).
4. Реалізувати метод обчислення об'єму конуса.
5. Розробити підсистему безпеки "Teacher" (у файлі `main.cpp`), яка за допомогою макросу `__FILE__` перевіряє правильність шляху до папки проєкту. У разі виявлення порушень регламенту згенерувати 100 звукових сигналів попередження (`\a`).
6. Реалізувати модуль автоматичного Unit-тестування класу з фіксацією результатів роботи програми у текстовому протоколі `TestResults.txt`.
7. Оформити репозиторій на GitHub з покроковою історією розробки.
