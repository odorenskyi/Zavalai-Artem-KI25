# Лабораторна робота № 12

**Тема:** Розробка абстрактних типів даних. Створення користувацьких класів у мові програмування C++

**Мета:** Навчитися проектувати та реалізовувати абстрактні типи даних у вигляді класів; опанувати механізми інкапсуляції, контролю доступу (public/private), написання конструкторів та методів; реалізувати автоматизоване модульне тестування (Unit Testing) та підсистему безпеки "Teacher" для перевірки цілісності структури репозиторію.

**Завдання:** 1. Створити клас `ClassLab12_Zavalai` для представлення геометричної фігури "Конус" у файлах `ModulesZavalai.h` та `ModulesZavalai.cpp`. Забезпечити фільтрацію від'ємних значень радіуса та висоти та реалізувати метод обчислення об'єму.
2. Розробити підсистему "Teacher" у файлі `main.cpp`, яка перевіряє правильність шляху до папки проєкту через макрос `__FILE__`. При порушенні — згенерувати 100 звукових сигналів попередження (`\a`).
3. Реалізувати модуль автоматичного Unit-тестування класу з фіксацією результатів у текстовому протоколі `TestResults.txt`.
