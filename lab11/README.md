<div align="center">
  <h1>Лабораторна робота № 11</h1>
</div>

**Тема:** Програмування задач із використанням динамічних структур даних (лінійних однозв'язних списків) та бінарних файлів у багатомодульних застосунках С++.

**Мета:** Опанувати теоретичні основи та отримати практичні навички роботи з динамічною пам'яттю, розробки алгоритмів створення та обробки однозв'язних лінійних списків, проєктування багатомодульних програм, а також збереження і зчитування структур даних за допомогою бінарних файлів мовою C++.

**Завдання:** Спроєктувати та розробити програмний модуль для управління базою даних «Довідник Укрпошти» (індекс, область, район, населений пункт, відділення поштового зв'язку). Дані під час виконання програми мають зберігатися в оперативній пам'яті у вигляді лінійного однозв'язного списку. 

**Варіант №:** 3
