# Лабораторна робота № 10

**Тема:** Обробка текстових файлів та рядкових величин у мові програмування C/C++. Створення модульних програм.

**Мета:** Отримати та закріпити практичні навички роботи з текстовими файлами (читання, запис, дозапис), засвоїти методи обробки символів та рядків (пошук, зміна регістру), а також навчитися проводити модульне тестування розробленого програмного забезпечення за допомогою тест-кейсів.

**Завдання:**
1. **Завдання 10.1:** Зчитати текст із файлу `input.txt`. Підрахувати кількість символів нижнього регістру та перевірити наявність заданого слова у тексті. Результати вивести у файл `output.txt`.
2. **Завдання 10.2:** Зчитати рядок (слово) з файлу `input.txt`. Змінити регістр кожного символу на протилежний. Виконати дозапис зміненого слова разом із поточною датою та часом у файл `output.txt`.
3. **Завдання 10.3:** Реалізувати обчислення математичної функції згідно з варіантом від введених параметрів `x, y, z`. Здійснити переведення цілого числа `b` у двійкову систему числення.

**Варіант №: 10**
